平板远程串流方案

2022.1.24

对于一个臭打游戏的大学生, 每天背着 5 斤左右的游戏本+充电器来回教室与宿舍, 不在插座旁边就活不下去的日子是不好受的. 如果能把游戏本放在宿舍, 只携带显示器与键鼠在教室远程操控电脑就好了.

经过学习网上资料, 笔者成功实现了这一方案.

购置平板电脑

平板电脑可以很好充当显示器的角色, 它的优点有轻便, 续航较强, 可以串流, 价格可以不高 (后两点与便携式显示器对比).

同时, 平板只需承担串流 (播放游戏本实时画面) 的负载, 对性能的要求不高, 可以砍预算. 它只要大概满足续航强, 轻便, 屏幕素质和游戏本差不多即可.

最后笔者摸了摸自己薄薄的钱包, 以 879 元的价格拿下了大厂平板中最便宜的一款: 联想小新 pad 4+64. 相比于两千到上万不等的 ipad, matepad 等平板, 小新 pad 确实只是个弟弟, 但它能串流, 只有 1 斤, 有7700mAh的电量, 有着比15.6英寸游戏本略小的11英寸, 2000*1200 的”2k” 屏, 已经可以满足笔者的需求.

配置软件

游戏本

  • 需要有 N 卡, 安装 Geforce Experience, 在”设置”->”SHIELD” 中打开 GAMESTREAM, 点击”添加”, 选取C:\Windows\System32\mstsc.exe
  • 打开 https://my.zerotier.com/network, 注册账号, create a network, 记住 Network ID 并保密, Name 处命名, Access Control 选择 PRIVATE, IPv4 Auto-Assign 选择 10.147.17.*. 打开 Download, 下载 Windows 的 ZeroTier, 打开 Control Panel, 输入 Network ID, Join Network. 回到网页, Members 处勾选 Auth?

平板

  • 打开 https://www.zerotier.com/download/ 下载 Android 的 ZeroTier, 打开, ADD NETWORK, 输入 Network ID, Add Network. 回到网页, Members 处勾选 Auth?, 至此游戏本与平板都连接到同一个 “Network” 内. 若游戏本与平板不连接同一个 WIFI, 需要打开 ZeroTier 的开关以打开 VPN, 否则不需要.

  • 打开 https://github.com/moonlight-stream/moonlight-android/releases, 下载 app-nonRoot-release.apk 并安装, 打开, 可以看到游戏本的带锁标识, 点击, 弹出验证码, 于游戏本输入, 成功串流.

其他

  • 串流信号 (视频) 分辨率为游戏本主屏幕的分辨率 a, 如果与平板的分辨率 b 一致, 最好; 如果分辨率 a > 分辨率 b, 也挺好, 可以在游戏本打开 NVIDIA Control Panel, 自定义出分辨率 b; 如果分辨率 a < 分辨率 b, 平板会有黑边或需要拉伸画面, NVIDIA Control Panel 也无法自定义出分辨率 b, 这时需要购买 HDMI 欺骗器, 需要骗出分辨率 b 以上的分辨率, 再用NVIDIA Control Panel 自定义出分辨率 b, 再到平板 moonlight 设置响应分辨率, 就可以 100% 利用平板屏幕了.
  • 平板 879, 无线键盘 37, 保护套 27, 支架 27, 欺骗器 5, 数据线转type-c 2, 方案总计花费 977 元.
  • 除了平板, 安卓手机也可以用相同方法实现串流. 手机屏幕虽小, 却也有妙用: VR. 眼镜还在路上, 功能尚在开发, 敬请期待.
